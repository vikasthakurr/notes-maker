## 🚀 Key Capabilities

- **🧬 Semantic Contextualization**: Generates content that balances deep technical logic (pointers, code, diagrams) with fluid, high-level structural explanations (ELI5 & Analogies).
- **🎨 Elite Presentation Mode**: Instantly builds "Gamma-style" interactive slide decks with optimized visual pacing and technical accuracy.
- **🖼️ Real-Time Intelligence Integration**: Autonomous agents perform real-time image retrieval (Google Search/Unsplash) to provide visual context during generation.
- **💎 Premium Design System**: A high-end, dark glassmorphic UI designed for elite developers, featuring smooth transitions and low-latency interaction.

---

## 🧠 Multi-Agent Orchestration

NoteForge utilizes a specialized agentic pipeline to ensure high-fidelity output:

- **The Planner (Architect)**: Blueprints the structural integrity of your notes or slides, ensuring logical flow and comprehensive coverage.
- **The Content Agent (Synthesizer)**: Populates the blueprint with crisp, precise technical data, balanced explanation paragraphs, and rich analogies.
- **The Image Agent (Illustrator)**: Programmatically retrieves and proxies technical illustrations to resolve CORS and ensure high-res rendering.

## 🛠️ Quick Start

### 1. Requirements
- **Node.js**: v18.0.0+
- **Cloud API Key**: [Get your Gemini 2.5 Flash Key](https://aistudio.google.com/app/apikey)

### 2. Setup
```bash
# Clone the core repository
git clone https://github.com/vikasthakurr/notes-maker.git

# Navigate and install dependencies
cd notes-maker
npm install
```

### 3. Launch
```bash
# Start the production-ready server
npm start
```
Access the environment at `http://localhost:3000`.
